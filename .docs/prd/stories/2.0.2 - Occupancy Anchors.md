# 2.0.2 - Occupancy Anchors

**Epic**: `2 - Content Quality & Scaling`

## Goal
Force the model to fill the white space and solve the "floating subject" issue by enforcing a "Macro/Filled" composition in the prompt templates.

## Scope
- **Update `prompt-manager.ts`**:
    - Modify the `getVariantPrompt` function to use the new "Macro-First" template:
      * *From*: `${tone} ${type} in a ${setting} ${action}`
      * *To*: `${tone} ${type} ${action}, closely surrounded by ${setting}, filling the frame, macro view.`

## Acceptance criteria
- [ ] `prompt-manager.ts` generates prompts using the new template structure.
- [ ] Generated prompts explicitly include "closely surrounded by", "filling the frame", and "macro view".
- [ ] Visually, the subject and setting occupancy targeting >80% of the canvas.

## Rollback
- Revert changes to `prompt-manager.ts`.
