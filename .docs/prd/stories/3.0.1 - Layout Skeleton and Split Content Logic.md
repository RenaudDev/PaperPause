**Epic**: `3 - Collection UX Overhaul`

## Goal
Restructure the collection list template to implement the Split Content pattern, separating header description, optional intro, gallery, and deep SEO article into distinct layout zones.

## Scope
- **Refactor `list.html`**:
  - Replace monolithic content block with structured zones:
    - **Header Zone**: Render `{{ .Description }}` frontmatter as page header text.
    - **Intro Zone**: Render `{{ .Summary }}` (content before `<!--more-->` delimiter) as optional intro below header.
    - **Gallery Zone**: Existing grid/gallery component (prepared for Expandable Vault in Story 3.0.2).
    - **Article Zone**: Render content after `<!--more-->` delimiter as deep SEO article below gallery.
  - Handle graceful fallback: if no `<!--more-->` delimiter exists, render full `.Content` in Article Zone.
- **CSS Updates**:
  - Add typography styles for header description and article zones.
  - Ensure responsive spacing between zones.

## Acceptance criteria
- [ ] `list.html` renders `description` frontmatter as page header distinct from page title.
- [ ] Content before `<!--more-->` renders as intro zone (if delimiter present).
- [ ] Content after `<!--more-->` renders as article zone below gallery.
- [ ] If no `<!--more-->` delimiter, full content renders in article zone without error.
- [ ] Layout is responsive and visually balanced on mobile and desktop.

## Rollback
- Revert changes to `themes/visual-sanctuary-theme/layouts/_default/list.html` and related CSS files.
